Standard Pen

The standard pen type is the default pen for all links not specified as one of the below types.

Lag Pen

Description:

A LAG link is a special kind of link between two network switches for efficiently passing network packets between them.

Line Style:

Manual Pen

Description:

A Manual link is a link created manually by a user, usually because they know of a network link in their network that was not automatically discovered by UVX.

Line Style:

Associated Pen

Description:

Some devices are controlled by other devices known as "controllers" (e.g., IP phones are managed by an IP phone controller, wireless access points are managed by a wireless access point controller, etc.) An Associated link is a link between a device its "controller" device.

Line Style:

Multi Pen

Description:

Multi links are links that involve multiple devices. Most links are point-to-point between just two devices, but some links involve more than two devices. Multi links are drawn in a "hub with spokes" fashion. There is one icon at the center that represents the link as a whole (the "hub"), and all the devices that participate in the link are drawn around the hub icon (these are the "spokes"). Therefore, a multi link consists of multiple edges between participating nodes (devices).
